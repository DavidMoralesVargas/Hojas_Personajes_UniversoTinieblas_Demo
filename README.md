<h2>Descripción del proyecto: </h2>

Este proyecto es una versión **DEMO** de un sistema de creación y gestión de Hojas de Personaje para el juego de rol **Vampiro La Mascarada** en el Universo de Tinieblas
El objetivo principal es **explorar y validar funcionalidades básicas** de lectura y visualización de hojas, así como el manejo de datos relacionados con clanes, atributos, disciplinas y otros elementos propios del sistema de juego.

**Este proyecto se realizó con fines de pruebas y práctica** por lo que todas las funcionalidad no están en su totalidad

Este proyecto no representa un producto final, sino un prototipo funcional orientado a aprendizaje, pruebas técnicas y validación del modelo de datos.

**Este proyecto fue pensado para:**
- Pruebas de concepto
- Desarrollo académico y personal
- Experimentación con tecnologías backend y frontend
- Iteración futura hacia una versión más completa
